# blinkist-config-python



## Contributing

Bug reports and pull requests are welcome on GitHub at https://github.com/blinkist/blinkist-config-python.

To install development requirements

```bash
pip install -r requirements/dev.txt
```

To run tests

```bash
pytest
```
